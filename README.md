## ✨ Fitur

### 📝 JSON Formatter
- Format JSON (pretty print)
- Minify JSON
- Copy output

### 🔍 Regex Tester
- Test regex pattern
- Highlight matches
- Support flags (g, i, m)

### 🎨 Color Converter
- HEX → RGB → HSL
- Color picker
- Copy format

### 🔐 Encoder/Decoder
- Base64 encode/decode
- URL encode/decode
- HTML encode/decode

### 📐 CSS Grid Generator
- Generate CSS Grid
- Visual preview
- Copy CSS code
